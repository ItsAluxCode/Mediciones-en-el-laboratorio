# 📏 Mediciones en el Laboratorio

Proyecto de análisis estadístico de mediciones físicas obtenidas en laboratorio,
con visualización de datos mediante gráficos de dispersión en Python.

## 📊 Descripción

Este proyecto genera gráficos de dispersión a partir de datos de medición
recolectados en laboratorio, permitiendo visualizar la distribución y variabilidad
de las medidas de **diámetro** y **longitud** de muestras analizadas.

## 🛠️ Tecnologías utilizadas

- **Python 3.x**
- **Matplotlib** – Visualización de datos
- **NumPy / Pandas** – Manejo de datos (según corresponda)

## ▶️ Cómo ejecutar

1. Clona el repositorio:
```bash
   git clone https://github.com/ItsAluxCode/Mediciones-en-el-laboratorio.git
   cd Mediciones-en-el-laboratorio
```

2. Instala las dependencias:
```bash
   pip install matplotlib numpy pandas
```

3. Ejecuta los scripts:
```bash
   python GraficoDiametro.py
   python GraficoLongitud.py
```
